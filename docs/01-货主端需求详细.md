# 货主端需求详细文档

## 概述
货主端是为需要运输货物的个人或企业用户设计的应用端，用户可以发布货运订单、选择司机、追踪货物、完成支付和评价。

---

## 一、用户注册与登录模块

### 1.1 注册流程
**页面：注册页**

#### 输入项：
- [ ] 手机号输入框（11位手机号验证）
- [ ] 验证码输入框（6位数字）
- [ ] 获取验证码按钮（60秒倒计时）
- [ ] 同意用户协议和隐私政策（勾选框）

#### 业务逻辑：
1. 点击"获取验证码"按钮
2. 后端发送短信验证码到手机
3. 用户输入验证码
4. 验证通过后自动登录，跳转到首页

#### 验证规则：
- 手机号格式验证（正则表达式）
- 验证码有效期：5分钟
- 同一手机号 1 分钟内只能发送 1 次验证码
- 同一手机号 1 小时内最多发送 5 次验证码

---

### 1.2 登录流程
**页面：登录页**

#### 登录方式：
- [ ] **验证码登录**（主要方式）
  - 手机号 + 验证码
  - 流程同注册
  
- [ ] **密码登录**（可选功能）
  - 手机号 + 密码
  - 忘记密码功能

#### 自动登录：
- [ ] 记住登录状态（Token 保持 7 天）
- [ ] 下次启动自动登录

---

### 1.3 实名认证
**页面：实名认证页**

#### 个人认证：
- [ ] 真实姓名输入
- [ ] 身份证号码输入
- [ ] 身份证正面照片上传
- [ ] 身份证反面照片上传
- [ ] 人脸识别验证（调用第三方 SDK）

#### 企业认证（可选）：
- [ ] 企业名称
- [ ] 统一社会信用代码（营业执照号）
- [ ] 营业执照照片上传
- [ ] 法人姓名
- [ ] 法人身份证号
- [ ] 企业联系电话

#### 认证状态：
- 未认证
- 认证中（审核中）
- 已认证
- 认证失败（显示失败原因，可重新提交）

#### 业务规则：
- 个人认证通过后才能发布订单
- 企业认证通过后可使用发票功能
- 认证信息一旦通过，不可修改（需联系客服）

## 二、发布订单模块

### 2.2 发单表单页
**页面：发布订单页**

##### 货物类型选择（单选）：
- [ ] 建材（砖、水泥、钢材等）
- [ ] 食品（生鲜、食品等）
- [ ] 家电（家具、电器等）
- [ ] 化工（化学品、涂料等）
- [ ] 数码（电子产品）
- [ ] 日用品
- [ ] 其他（需手动输入）

##### 货物详情：
- [ ] **货物重量**
  - 输入框 + 单位选择（公斤/吨）
  - 范围：0.1 ~ 50 吨
  
- [ ] **货物体积**
  - 输入框 + 单位选择（立方米）
  - 范围：0.1 ~ 100 立方米
  
- [ ] **货物数量**
  - 输入框 + 单位（件/箱/批）
  
- [ ] **货物描述**（可选）
  - 多行文本输入框
  - 最多 200 字
  
- [ ] **货物照片**（可选）

  - 支持拍照或从相册选择
  - 每张图片最大 5MB

##### 特殊需求：
- [ ] 是否需要装卸工（开关按钮）
  - 如果选是，显示装卸工数量选择（1-5人）
  
- [ ] 是否需要包装（开关按钮）

- [ ] 是否需要保险（开关按钮）

#### 第二步：地址信息

  - 可选择常用联系人
  
- [ ] **备注**（可选）
  - 如"仓库在 3 号门"
  - 最多 50 字

##### 卸货地址：

- [ ] 支持从常用地址选择
- [ ] 支持"复制装货地址"快捷按钮（适用于同城往返）

- [ ] 保存为常用地址（勾选框）
- [ ] 地址别名（如"公司"、"仓库A"）

##### 车型选择（单选）：
- [ ] 小货车（1.5吨以下，车长 3-4 米）

- [ ] 大货车（5-10吨，车长 6-9 米）
- [ ] 厢式货车（带箱体）
- [ ] 平板车
- [ ] 其他（自定义）

##### 车长要求（可选）：
- [ ] 下拉选择：3米 / 4米 / 6米 / 9米 / 不限

##### 特殊要求：
- [ ] 是否需要尾板（开关按钮）
- [ ] 是否需要篷布（开关按钮）

---

#### 第四步：时间要求

##### 装货时间选择：
- [ ] **立即用车**（单选项）
  - 30 分钟内到达
  
- [ ] **预约用车**（单选项）
  - 日期选择器
  - 时间选择器（半小时为单位）
  - 最早：当前时间后 2 小时
  - 最晚：7 天后

- [ ] 日期选择器
- [ ] 时间选择器
- [ ] 或选择"尽快送达"

---

#### 第五步：价格设置

##### 定价模式（单选）：
- [ ] **系统估价**（推荐）
  - 根据距离、车型、货物重量自动计算
  - 显示价格范围（如 ￥200 - ￥250）
  - 实时更新
  
- [ ] **一口价**
  - 货主自己输入价格
  - 输入框（￥）
  - 范围：50 - 50000 元
  
- [ ] **议价模式**
  - 不设置价格
  - 让司机报价，货主选择

##### 价格明细（系统估价模式下显示）：
- 基础运费：￥ XX
- 距离费用：￥ XX（XX 公里 × XX 元/公里）
- 装卸费：￥ XX（可选）
- 其他费用：￥ XX
- **总计：￥ XXX**

##### 支付方式选择：
- [ ] 在线支付（微信/支付宝）
- [ ] 货到付款（现金/扫码）

#### 第六步：确认发布

### 3.3 订单操作功能

#### 修改订单：
- 仅在"待接单"状态可修改
- 可修改内容：
  - 装货/卸货地址
  - 装货时间
  - 货物信息
  - 价格
- 修改后重新发布

#### 取消订单：
- **待接单状态**：无违约金
- **待装货状态**：扣除订单金额的 10% 作为违约金
- **运输中状态**：不允许取消（需联系客服）
- 取消原因选择：
  - 计划变更
  - 找到其他车辆
  - 价格不合适
  - 其他（手动输入）

#### 追加小费：
- 在"运输中"或"待确认"状态可追加
- 金额选择：10元 / 20元 / 50元 / 自定义
- 即时到账司机账户

#### 上报异常：
- 异常类型选择：
  - 货物损坏
  - 货物丢失
  - 司机延误
  - 司机态度恶劣
  - 地址错误
  - 其他
- 上传证据照片（最多 9 张）
- 文字描述（最多 500 字）
- 提交后，客服介入处理

## 四、支付与结算模块

### 4.1 支付方式

#### 在线支付：
- [ ] **微信支付**
  - 调用微信支付 SDK
  - 支持余额、银行卡、信用卡
  
- [ ] **支付宝支付**
  - 调用支付宝支付 SDK
  
- [ ] **银行卡支付**（可选）

#### 货到付款：
- 司机送达后，货主扫描司机收款码
- 或现金支付
- 货主确认收货后，订单完成

---

### 4.2 支付流程

#### 在线支付流程：
1. 货主发布订单 → 冻结资金（不扣款）
2. 司机接单 → 资金继续冻结
3. 司机送达，货主确认 → 自动扣款，打款给司机
4. 如果取消，冻结资金自动退回

#### 资金托管机制：
- 订单金额暂存平台账户
- 平台抽成（如 5%）
- 司机收入 = 订单金额 × 95%
- 订单完成后，自动分账
